IPPcode2019github

A set of scripts for analysis and interpretation nonstructured, imperative language IPPcode19.

It is split into 2 files, a parse.php file that converts the code into an XML representation and interpret.py that interprets the code. Unfortunatelly the readme exists only for the second part.

IPK-HTTP-servergithub

A basic HTTP server written in C. Responds to 3 request messages:

  • /hostname - returning the hostname.
  • /cpu-name - returning the name of the server cpu.
  • /load - returning the current cpu load of the server.
