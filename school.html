IFJcode2018 language Compiler inc Cgithub

A compiler of custom language IFJcode2018.

This is probably the first bigger group project you typically do at the shool and one of its harder ones. In the group of 4 people my main contributions were the lexical analyser and some work on the parser and semantics analysis.

Unfortunatelly the code generation is not finished but other than that, all parts of the analysis and parsing are functioning well.

IPPcode2019 language Interpret in Python and PHPgithub

A set of scripts for analysis and interpretation nonstructured, imperative language IPPcode19.

It is split into 2 files, a parse.php file that converts the code into an XML representation and interpret.py that interprets the XML. Unfortunatelly the readme exists only for the second part.

Basic HTTP Server in Cgithub

Responds to 3 request messages:

  • /hostname - returning the hostname.
  • /cpu-name - returning the name of the server cpu.
  • /load - returning the current cpu load of the server.
